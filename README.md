## Installation

Install the latest [Node.js](https://nodejs.org/) to the system globally.
Clone package:
```
git clone https://github.com/Bitvinus/frontend-project-lvl1.git
make install-local
```
To install a package:
- Go to project directory.
- Run command:
	```sh
	$ npm link
	```
